# /issues - Find GitHub Issues

Search through existing issues using the gh CLI to find matching issues.

## Usage

`/issues <search query>`

## Instructions

1. Use `gh issue list` and `gh issue search` to find matching issues
2. Consider matches by:
   - Similar titles or descriptions
   - Same error messages or symptoms
   - Related functionality or components
   - Similar feature requests

## Output

List matching issues with:
- Issue number and title
- Brief explanation of why it matches
- Link to the issue

If no clear matches are found, say so.
